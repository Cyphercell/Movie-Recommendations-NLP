 Movie Recommendation System using NLP

This project is a **content-based movie recommendation system** built using **Natural Language Processing (NLP)** techniques.  
It recommends movies based on similarity between movie descriptions and metadata using IMDb data.

---

 Features
- Content-based movie recommendations
- NLP text preprocessing and vectorization
- Cosine similarity for recommendation logic
- Interactive **Streamlit web app**
- Uses IMDb Top 1000 movies dataset

---

 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- NLP (TF-IDF / Count Vectorizer)
- Streamlit
- Pickle
